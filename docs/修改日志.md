### 2026-06-12：运行过程窗口布局改造

**目标**：将运行过程输出窗口从内嵌式底部面板改为弹出式独立窗口。

原布局：显示时在主窗口下方20%，隐藏时在导航栏下方20%显示小窗口。
新布局：显示时在主窗口左下角弹出独立窗口（可调整大小、可调透明度），隐藏时不显示任何内容。

**修改内容**：

#### 1. 删除 sub_output_frame（导航栏下方小窗口）
- 删除了 L9154-L9165 的 sub_output_frame 创建代码
- 删除了 L9149-L9150 的重复 output_visible 读取
- left_layout 中 nav_panel 不再有 8:2 的比例
- 删除了导航折叠/展开中所有 sub_output_frame 引用

#### 2. 替换 output_frame（右侧底部面板）为 QDialog 弹出窗口
- 删除了 L9202-L9222 的 output_frame 创建代码
- 新增 _output_dialog（QDialog），作为独立的顶层弹出窗口
- 窗口标志：Window | CustomizeWindowHint | WindowCloseButtonHint | WindowStaysOnTopHint | WindowTitleHint
- 设置了 setSizeGripEnabled(True) 和 setWindowModality(NonModal)
- 保留了 main_horizontal_layout.addLayout(right_layout, 1) 这行不删
- 输出文本控件改为 _output_text

#### 3. 新增方法
- _position_output_dialog(): 将弹窗定位到主窗口左下角
- _apply_output_dialog_opacity(): 从配置文件读取透明度并应用到弹窗

#### 4. 修改已有方法
- on_toggle_output: 控制弹出窗口 show/hide，不再操作内嵌面板
- append_output: 简化，只写入 _output_text，去除子窗口过滤逻辑
- on_clear_output: 简化，只清除 _output_text
- update_output_window_size: 现在实际调整弹窗大小（之前是空实现 pass）
- restore_user_runtime_state: 改为控制 _output_dialog 的 show/hide

#### 5. 清理导航栏代码
- 折叠展开中的 sub_output_frame.setVisible 替换为注释
- toggle_output_btn 和 clear_btn 保留在导航栏

**涉及的配置项**：
- output_window_size: 输出窗口大小 [宽度, 高度]（设置页滑块控制）
- output_opacity: 输出窗口透明度（0-100，新增配置）

### 2026-06-12：输出窗口优化（无边框筛选器式弹窗 + 行为改进）

**目标**：优化输出窗口的交互体验：无边框筛选器样式、弹窗位置、拉伸调整、透明度控制、页面跟随、焦点行为。

**修改内容**：

#### 1. 窗口样式改为筛选器式无边框弹窗
- 移除传统窗口标题栏和边框（FramelessWindowHint）
- 添加圆角白色容器（border-radius: 8px）和阴影效果（QGraphicsDropShadowEffect）
- 自定义标题栏含"运行输出"标签（后因重复移除所有标题栏按钮）
- 添加 QSizeGrip 底部右下角尺寸手柄

#### 2. 输出窗口按钮布局重构
- 透明度按钮从弹窗标题栏移至左侧导航栏（在"隐藏"和"清除"按钮中间）
- 移除弹窗内所有按钮（运行输出标签、清除、关闭）→ 全由导航栏控制
- 透明度按钮折叠导航时自动隐藏/显示

#### 3. 焦点与隐藏行为
- 移除 WindowStaysOnTopHint → 不再覆盖其他应用窗口
- 添加全局事件过滤器（安装于 QApplication），点击弹窗外区域自动隐藏
- 切换页面时自动隐藏输出窗口（switch_to_page 顶部）
- 点击输出控制按钮（透明度/清除/显示）不触发外部点击隐藏
- 启动时始终默认隐藏窗口（output_visible = False）

#### 4. 弹窗位置改为"隐藏"按钮上方
- _position_output_dialog() 用 mapToGlobal 获取隐藏按钮屏幕坐标
- 弹窗出现在按钮正上方（间距 4px），左右不超过屏幕边界

#### 5. 弹窗随主窗口移动/缩放
- 新增 MainFrame.moveEvent：主窗口移动时重新定位弹窗
- 更新 MainFrame.resizeEvent：主窗口缩放时重新定位弹窗

#### 6. 边缘拉伸调整
- 支持右边缘（↔ 光标）、上边缘（↕ 光标）、右上角（↖ 光标）拉伸
- 拉伸时使用 setGeometry 同时调整位置和大小（防止从下方变大）
- 事件过滤器改用 w.window() is self._dlg 检测，解决 QTextEdit 遮挡导致鼠标事件丢失的问题
- 坐标通过 mapFromGlobal 统一映射到对话框坐标系

#### 7. 大小与透明度持久化
- 新增 _save_output_dialog_size() 保存到 shared_settings.json 和 RuntimeState
- 新增 _cycle_output_opacity() 循环切换透明度（100% → 80% → 60% → 40%）
- 启动时优先从 RuntimeState（个人配置文件）读取窗口大小
- 设置页基础设置新增：输出窗口透明度/宽度/高度滑块（实时生效）

#### 8. 其他修复
- 修复 QTimer 作用域问题：在 eventFilter 顶部统一 import
- 修复 e.globalPosition() 仅鼠标事件可用的 AttributeError
- 清理所有乱码占位符（?????????）
- 统一窗口大小默认值为 [600, 300]
- 修复导航展开时透明度按钮不显示的问题

**涉及的配置项**：
- output_window_size: 输出窗口大小 [宽度, 高度]
- output_opacity: 输出窗口透明度（10-100）
- output_visible: 启动时输出窗口可见性（始终 False）
