## 总体架构

输出窗口是一个独立 QDialog，无边框筛选器式弹窗，通过 _OutFilter 事件过滤器控制边缘拉伸、外部点击隐藏、跟随移动。


## 一、窗口创建

### 1.1 窗口标志
MainFrame.__init__() 中，原带有 CustomizeWindowHint / WindowCloseButtonHint / WindowStaysOnTopHint / WindowTitleHint，改为：
- FramelessWindowHint（无边框）
- WA_TranslucentBackground（透明背景透阴影）

### 1.2 容器布局
原 popup_layout + QLabel("过程：") 替换为：
- _oc = QFrame：白底 + border-radius:8px + border:1px solid #D0D0D0
- _ov = QVBoxLayout：放 QTextEdit（透明背景 + border-radius:8px）
- QSizeGrip：右下角尺寸手柄
- _ol = QVBoxLayout（外边距6px为阴影留空间）
- QGraphicsDropShadowEffect：blur=24, offset=(0,4), color=rgba(0,0,0,60)

### 1.3 启动默认隐藏
self.output_visible = False  （替代原从 runtime_state 读取）


## 二、事件过滤器 _OutFilter

### 2.1 安装
_of = _OutFilter(dlg, main)
self._output_dialog.installEventFilter(_of)  # dialog 事件
QApplication.instance().installEventFilter(_of)  # 全局事件
self._output_dialog.setMouseTracking(True)

### 2.2 类核心逻辑
- __init__: _dlg, _main, _resizing, _resize_edge, _resize_start, _resize_start_geo, _save_timer, _edge_margin=10
- _edge(pos): 检测 mouse 是否在右/右上/上边缘 10px 内，返回 "r"/"tr"/"t"/None
- eventFilter 处理顺序:
  1. Resize + w is self._dlg -> 防抖500ms 保存尺寸
  2. MouseMove + on_dlg -> 拉伸/光标形状
  3. MouseButtonPress + on_dlg + Left -> 开始边缘拉伸
  4. MouseButtonRelease + on_dlg -> 结束拉伸
  5. MouseButtonPress + Left + 外部 -> widgetAt 排除按钮后隐藏

### 2.3 关键设计
- on_dlg: hasattr(w,"window") and w.window() is self._dlg 替代 w is self._dlg
- 坐标: self._dlg.mapFromGlobal(e.globalPosition().toPoint())
- setGeometry(x, y+dy, w, h) 替代 resize(w, h)，防止从下方变大
- 排除: QApplication.widgetAt(cp) 沿 parentWidget 链查三个控制按钮
- QTimer 在 eventFilter 顶部 import，避免作用域问题

## 七、常见问题

- 边缘拉伸不生效：on_dlg 用 w.window() + mapFromGlobal
- 从上方拉伸从下方变大：setGeometry 替代 resize
- 点击按钮触发隐藏：QApplication.widgetAt + parentWidget 链排除
- globalPosition() 报错：只在鼠标事件中计算
- QTimer 未绑定：在 eventFilter 顶部 import
- 按钮消失：toggle_navigation 加 setVisible
- 弹窗不跟随：moveEvent/resizeEvent 加 _position_output_dialog
